Project Report: Blockchain + AI Fraud Detection (End-to-End)

FastAPI Hardhat web3.py React + MetaMask Frontend Solidity Smart Contracts DeFi GenAI

Financial fraud continues to be a major global challenge, particularly in decentralised finance (DeFi). This project explores how combining blockchain technology with artificial intelligence (AI) can improve fraud detection and provide secure, transparent logging of suspicious transactions. The system integrates multiple technologies and models to build a complete fraud detection pipeline:

  • Uses blockchain for transparent and immutable fraud logging.
  • Applies AI models to detect suspicious financial transactions in real time.
  • Evaluates several machine learning approaches, including:
    • Ensemble models: Random Forest, XGBoost, LightGBM, CatBoost
    • Graph-based models: GCN, GraphSAGE, and Graph Attention Networks (GAT)
    • A custom Transformer architecture for advanced pattern detection
  • Models were trained on the PaySim dataset and optimized to prioritise fraud detection.
  • Performance was evaluated using Accuracy, Precision, Recall, F1 Score, and AUC.
  • Model behaviour and predictions were analysed using threshold tuning, SHAP explanations, and fraud-score visualisations.

To support real-world deployment, the project also includes a full-stack implementation that connects AI models with blockchain infrastructure.

  • A FastAPI backend ensures consistent model training and prediction pipelines.
  • Fraud predictions are stored initially locally in SQLite for testing and then logged on-chain through a Solidity smart contract.
  • Smart contracts are deployed using Hardhat and connected through web3.py.
  • A React interface with MetaMask allows secure wallet interaction and visualisation of fraud detection results.
  • System robustness was tested using:
    • Rule-based fraud or intrusion simulations
    • Generative adversarial techniques to create synthetic fraud patterns

Additional experiments analysed system performance challenges, including blockchain latency and large-scale dataset processing. Proposed improvements and further details are discussed in the detailed report provided below.

MATLAB Segmentation Classification Feature Extraction

Malignant melanoma is the most dangerous form of skin cancer, and early detection is essential for effective treatment. This project explores how machine learning and computer vision techniques can assist clinicians by improving the speed and consistency of skin lesion diagnosis.

  • Develops a MATLAB-based system to classify skin lesion images as malignant melanoma or benign.
  • Uses computer vision techniques combined with a Support Vector Machine (SVM) classifier.
  • The workflow includes several processing stages:
    • Image preprocessing
    • Lesion segmentation
    • Feature extraction
  • Extracts shape, colour, and texture features from segmented lesions.
  • Uses the extracted features as input for the SVM classification model.
  • Evaluates system performance using:
    • Accuracy
    • Sensitivity
    • Specificity
  • Provides visual examples of intermediate processing steps to demonstrate how the system analyses images.
  • Shows how classical image processing and machine learning techniques can support automated skin cancer screening and assist early diagnosis.
